# 粉笔摸鱼浏览器

一个受墨鱼阅读启发的 Electron 桌面小工具，默认打开粉笔网，适合在半透明窗口中浏览和刷题。

## 功能

- 默认加载 `https://www.fenbi.com/`
- 无边框透明窗口与透明度滑块
- 窗口置顶、居中、后退、前进、刷新、回到粉笔首页
- 老板键：`Ctrl/⌘ + Shift + H` 隐藏/恢复，`Ctrl/⌘ + Shift + Q` 退出
- 鼠标离开窗口时自动弱化，回到工具栏时恢复
- 可选鼠标穿透模式，方便把窗口当半透明参考层

## 开发

```bash
npm install
npm start
```

## 检查

```bash
npm run check
```
